# Wandelplan 6 maanden voor iPhone agenda

Startdatum: maandag 6 juli 2026
Tijdstip wandelingen: 19:30
Tijdzone: Europe/Amsterdam

## Bestanden

- `wandelplan.ics`: de agenda-feed voor iPhone, Outlook en Google Calendar.
- `index.html`: kleine landingspagina voor GitHub Pages.

## Publiceren via GitHub Pages

1. Maak een nieuwe publieke GitHub repository aan, bijvoorbeeld `wandelplan`.
2. Upload deze bestanden naar de repository.
3. Ga naar Settings > Pages.
4. Kies als source `Deploy from a branch`.
5. Kies branch `main` en folder `/root`.
6. Wacht tot GitHub Pages de site publiceert.

Je ICS-link wordt dan ongeveer:

`https://jouwgebruikersnaam.github.io/wandelplan/wandelplan.ics`

Op de iPhone kun je deze toevoegen via:
Instellingen > Agenda > Accounts > Nieuwe account > Andere > Voeg agenda-abonnement toe.

Gebruik daar de HTTPS-link naar `wandelplan.ics`.
